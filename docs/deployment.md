# Інструкції для release engineer/DevOps з розгортання проекту
## 1. Вимоги до апаратного забезпечення
    - Мінімум 4ГБ оперативної пам’яті
    - Двох або краще 4-ядерний процесор (не старіший за Intel Core i5 або AMD Ryzen 5) з тактовою частотою не менше 1.6ГГц
    - Накопичувач (краще SSD) з вільним місцем не менше 5.5ГБ
## 2. Необхідне програмне забезпечення
    - Дистрибутив PostgreSQL
    - Клієнт СКБД DBeaver (опційно)
    - IDE Pycharm
    - Операційна система Windows не старіше 10 версії (64-бітна версія)
## 3. Для перенесення коду з репозиторію в IDE, необхідно просто клонувати репозиторій.
## 4. Перевірка працездатності коду
- Додаток має статус-бар у лівому нижньому куті; якщо запуск відбувся успішно, в ньому з'явиться повідомлення «Готово до роботи». Для повної впевненості можна спробувати перевірити роботу основних функцій. Якщо нічого не падає, то код повністю робочий.
