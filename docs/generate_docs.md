# Інструкція з генерації документації до коду
## 1. Відкрити термінал.
## 2. Перейти з папки проекту в папку docs за допомогою команди `cd docs`.
## 3. Виконати команду `.\make.bat html`. Після виконання команди, папка `build` буде створена автоматично. 
## 4. Після завершення генерації в термінал буде виведено повідомлення `The HTML pages are in build\html.`
## 5. Для того, щоб відкрити головну HTML - сторінку: 
    * 1. Знайдіть в IDE вкладку "Project".
    * 2. Перейдіть до папки "build/html/".
    * 3. Відкрийте файл "index.html".
## P.S. Для генерації нової документації не потрібно видаляти стару. <br> Просто запустіть команду генерації; старі файли буде замінено автоматично.
